# Google-Playstore-E.D.A
Exploratory Data Analysis on Google Play Store dataset using Python
Google Play Store Apps – Exploratory Data Analysis

## Objective
Analyze Google Play Store app data to understand:
- Free vs Paid app performance
- Category-wise distribution
- Factors influencing installs and ratings

##  Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Insights
- Free apps dominate in installs
- Paid apps show slightly higher average ratings
- Certain categories contribute most to total installs

- playstore eda.ipynb – complete EDA notebook
